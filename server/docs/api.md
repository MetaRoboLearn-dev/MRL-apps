# User
| Akcija | Path                                  | Params / Body                                                                                                                                                                                      | Return                               | Opis                                      |
|--------|---------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------|-------------------------------------------|
| GET    | `/api/users/<user_id:int>`            | /                                                                                                                                                                                                  | User                                 | Vraća User-a po njegovom ID-u             |
| GET    | `/api/users`                          | `skip=int` – offset  <br> `limit=int` – gornja granica <br> `role_id=int` <br> `active_only=bool` <br> `search=string` – pretraga po username, first_name, last_name <br> `order_by_username=bool` | list[User]                           | Vraća listu User-a                        |
| GET    | `/api/users/exists/<username:string>` | /                                                                                                                                                                                                  | `{ exists: bool, username: string }` | Provjerava postoji li User s tim username |
| POST   | `/api/users`                          | `username=string` <br> `password_hash=string` <br> `first_name=string` <br> `last_name=string` <br> `role_id=int`                                                                                  | User                                 | Stvara novog User-a                       |
| POST   | `/api/users/<user_id:int>/deactivate` | /                                                                                                                                                                                                  | User                                 | Deaktivira User-a po njegovom ID-u        |
| POST   | `/api/users/<user_id:int>/activate`   | /                                                                                                                                                                                                  | User                                 | Aktivira User-a po njegovom ID-u          |
| PATCH  | `/api/users/<user_id:int>`            | `username=string` <br> `password_hash=string` <br> `first_name=string` <br> `last_name=string` <br> `role_id=int`                                                                                  | User                                 | Ažurira podatke User-a                    |
| DELETE | `/api/users/<user_id:int>`            | /                                                                                                                                                                                                  | `{ deleted: true }`                  | Briše User-a po njegovom ID-u             |
